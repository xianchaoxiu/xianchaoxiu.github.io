课题组<mark>"以优化算法为核心、交叉应用为导向"</mark>。
培养上秉持理论与实践并重, 学生可依兴趣与规划灵活选题, 或深耕理论前沿, 或聚焦产业实践。
每周组会复盘进展, 每月咖啡会疏导困惑, 每人支持1-2次国内学术会议。允许研二暑假实习, 求职季预留充足时间。
对学术常怀敬畏之心, 若仅以混学位为目标, 恐难契合。愿各位在科研中求索真知, 不负研究生的宝贵韶华


### 什么是研究生
* 本质上, 研究生是学习从事科研的学生
* 研究生不是人才, 优秀的研究生才是人才
* 研究生与本科生的区别
  - 研究生: 体验知识 → 学会创造 → 开展研究 → 提升素养 → 增长能力
  - 本科生: 学习 → 考试 → 60分 → 毕业 → 就业


### 什么是导师
* 导师目前只是合作者, 有可能发展成朋友
* 导师与学生双向选择, 双方都有权解除师生关系
* 导师与学生认识问题的差异
  - 导师: 努力刻苦钻研 → 提高科研能力 → 夯实学术水平 → 毕业
  - 学生: 毕业 → 必要的水平 → 必要的能力 → 必要的努力

 
  
### 应具备的基本素质
* 自驱力强, 身心健康
* 先学做人, 再谈做研究
  - 不要不懂装懂
  - 不要认为别人都是傻瓜
  - 不要认为天上会掉馅饼
  - 不要认为别人都会以你为中心
  - 不要把偶然性的侥幸当做必然性的成功
  - 不要在同一个地方跌倒两次或两次以上
  - 不要把别人的宽容和善意当成是软弱可欺
* 混学位学生语录
  - 老师, 什么最容易做又能毕业 → 不想努力
  - 老师, 听师兄师姐们说没有文章也能毕业 → 不想写文章
  - 老师, 来你办公室几次你都不在 → 耍小聪明
  - 老师, 下次我一定注意 → 还有下次
  - 老师, 我最近情绪不好 → 你别逼我


### 如何做研究
* 怎样读书
  - 读大师的书和论文, 你才可能成为大师
  - 读垃圾只能产生垃圾，除非你能变废为宝
  - 以教科书入门, 以专著进阶, 以综述拓展视野, 以期刊跟踪前沿
* 怎样选题
  - 初期不必追求具体, 随着研究推进, 选题会自然聚焦
  - 简单问题易上手但价值有限, 复杂问题有深度但门槛高
  - 一个成功的选题, 往往意味着研究已完成50%甚至更多
* 怎样读文献
  - 泛读: 了解问题和结论, 判断其是否值得细读
  - 精读: 提出问题 → 寻找方法 → 结果验证
  - 反思: 创新在哪里 → 真的创新吗 → 能推广吗
  - 借鉴: 主动学习其行文逻辑与表达风格, 把好句子记录下来
* 常见表达误区
  - 照搬自己不懂的概念, 企图让别人懂 → 那是不可能的
  - 自己创造新概念, 不加解释以为别人也懂 → 打哑谜
  - 将自己不懂得东西, 装作很轻松地表述为显然 → 一点都不显然
  - 看完整篇文章, 不知作者做了什么 → 可能构成剽窃
  - 图表缺乏规范标注 → 只剩艺术价值
  - 同一个符号对应n种意思 → 你猜
