### 什么是研究生
* 学习从事科学研究的学生 
* 与本科生的区别
  - 本科生: 学习 → 考试 → 60分 → 毕业 → 就业
  - 研究生: 体验知识 → 学会创造 → 开展研究 → 提升素养 → 增长能力


### 基本素质
* 先学做人
* 科研内在驱动力
* 积极向上的人生理想


### 学风和做人
* 不要不懂装懂
* 不要认为别人都是傻瓜
* 不要认为天上会掉馅饼
* 不要认为别人都会以你为中心
* 不要把偶然性的侥幸当做必然性的成功
* 不要在同一个地方跌倒两次或两次以上
* 不要把别人的宽容和善意当成是软弱可欺


### 什么是导师
* 导师目前只是合作者, 有可能发展成朋友
* 导师与学生双向选择, 双方都有权解除师生关系
* 导师与学生认识问题的差异
  - 导师: 努力刻苦钻研 → 提高科研能力 → 夯实学术水平 → 毕业
  - 学生: 毕业 → 必要的水平 → 必要的能力 → 必要的努力

 
### 如何学习
* 主动学习
  - 老师, 我想XX, 您看是否可行
  - 老师, 我有不同的观点, 因为XX
* 混学位学生语录
  - 老师, 什么最容易做又能毕业 → 不想努力
  - 老师, 听师兄师姐们说没有文章也能毕业 → 不想写文章
  - 老师, 来你办公室几次你都不在 → 耍小聪明
  - 老师, 下次我一定注意 → 还有下次
  - 老师, 我最近情绪不好 → 你别逼我


### 如何做研究
* 怎样读书
  - 读大师的书和论文, 你才可能成为大师
  - 读垃圾只能产生垃圾，除非你能变废为宝
  - 以教科书入门, 以专著进阶, 以综述拓展视野, 以期刊跟踪前沿
* 怎样选题
  - 简单问题易上手但价值有限, 复杂问题有深度但门槛高
  - 初期不必追求具体, 随着研究推进, 选题会自然聚焦
  - 一个成功的选题, 往往意味着研究已完成50%甚至更多
* 怎样读文献
  - 泛读: 了解问题和结论, 判断其是否值得细读
  - 精读: 问题 → 方法 → 结果
  - 反思: 问题是特例吗, 能推广吗, 值得进一步考虑吗
  - 借鉴: 主动学习其行文逻辑与表达风格, 把好句子记录下来



### 学术论文写作
* 为什么我写的东西别人看不懂
  - 写给外星人 → 天书
  - 写给自己 → 隐私
  - 写给同行 → 论文
* 常见表达误区
  - 照搬自己不懂的概念, 企图让别人懂 → 那是不可能的
  - 自己创造新概念, 不加解释以为别人也懂 → 打哑谜
  - 将自己不懂得东西, 装作很轻松地表述为显然 → 一点都不显然
  - 看完整篇文章, 不知作者做了什么 → 可能构成剽窃
  - 图表缺乏规范标注 → 只剩艺术价值
  - 同一个符号对应n种意思 → 你猜  
   

### 讨论班
* 为什么参加
  - 应付差事
  - 给老师面子
* 核心价值
  - 分享研究进展, 探讨疑难问题
  - 学会表达自身观点, 理性回应他人质疑
  - 锻炼创造性思维, 从不同视角中获得启发
  - 提升基础科研技能, 包括PPT制作和成果展示
